\doxysubsection{Detailed Description}
Keyword-\/based intent classifier using string matching. 

This classifier uses a predefined dictionary of keywords to identify intents. It\textquotesingle{}s fast but limited to exact keyword matches. Works well for unambiguous commands with clear trigger words.
