\doxysubsection{Detailed Description}
Module for processing timer and countdown requests. 

Extracts duration entities from natural language. Handles commands like "{}set timer for 10 minutes"{}, "{}start countdown for 30 seconds"{}, "{}timer 5 minutes"{}.
