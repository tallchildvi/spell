\doxysubsection{Detailed Description}
Module for processing reminder creation requests. 

Extracts datetime entities and reminder text from natural language. Handles commands like "{}remind me to call mom tomorrow"{}, "{}set a reminder for 8am"{}, "{}reminder to buy milk"{}.
