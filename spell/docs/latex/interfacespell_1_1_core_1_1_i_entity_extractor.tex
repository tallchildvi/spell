Interface for entity extraction components in the NLP pipeline. 

Implementations extract structured entities (dates, numbers, units) from text and populate them into the \doxylink{classspell_1_1_core_1_1_intent_result}{Intent\+Result}\textquotesingle{}s Entities dictionary.
